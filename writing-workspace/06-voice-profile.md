# 6단계 — 문체 프로필 카드 (Voice & Persona)

## 문체 프로필

| 항목 | 분석 결과 |
|---|---|
| **어조** | 담담한 하십체(–다). 감정 고조 시 해라체 전환. 혼잣말에 독자가 엿듣는 구조 |
| **평균 문장 길이** | 20어절 (단문 35% / 장문 65%) |
| **은유 패턴** | 자연·물질(증류수, 커피, 탄산수, 복숭아) + 일상·직장(A대리, 의자 게임). 감각적 구체성이 특기 |
| **어휘 수준** | 순우리말 50% / 한자어 35% / 외래어 15%. 사유=한자어, 감각=순우리말, 직업=외래어 |
| **리듬감** | 3단 점층, 자기반박("그런데~이유는 뭘까"), 여운 결말(날씨·행동으로 감정 대신) |
| **글쓰기 성향** | 논리 중시 / 내면 소리대로 작성 / 구조는 쓰면서 잡힘 |

## 시그니처 (이 글에서 유지해야 할 작가의 DNA)

1. **감각 번역**: 추상을 즉각 물질로. "운명"→"산맥", "위기"→"추락", "황금기"→"태양이 가장 높을 때"
2. **칼날 단문**: 중요한 전환마다 짧은 단문 한 방. "그래프를 그려보았다." "맞아떨어졌다."
3. **자기반박 리듬**: 주장 → 스스로 의심 → 한 발 더 나아감. "그런데 그게 진짜일까. 지도를 펼쳐보았다."
4. **여운 결말**: 감정을 직접 말하지 않음. 행동이나 감각으로 마무리.
5. **자기비하 유머**: 무거운 주제를 가볍게 돌리는 타이밍. 독자가 웃다가 생각하게 만듦.

## 이 글을 위한 구체적 가이드라인 3개

1. **사주·명리 용어 등장 시 → 즉시 감각적 비유로 번역**
   - "목화통명" → "태양이 숲을 태우며 타오르는 것"
   - "대운" → "10년짜리 계절"
   - "정합성" → "지도와 발자취가 겹치는 순간"
   기술 용어를 그대로 두지 말고, 곧바로 자연·물질 비유로 이어 친근감을 살린다.

2. **본론의 대운 서사는 '칼날 단문 + 사색적 장문' 교차**
   - 대운 전환마다: "정묘. 다섯 살. 심장이 멈추었다." (단문)
   - 이어서: 장문으로 그 시기의 감각·기억을 끝까지 밀고 간다
   - 결코 설명하지 않는다 — 보여준다.

3. **결론부에서는 짧은 단문 3개로 점층 마무리**
   - 예: "지도는 그려져 있다. / 꼭지점을 눌렀다. / 걷는 건 나다."
   - 마지막 문장은 감각·행동으로 끝낸다 (날씨, 소리, 맛, 감촉)
