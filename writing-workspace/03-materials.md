## 이질적 재료 간 연결 고리

### 1. "사주 그래프는 동양의 Quantified Self다"
게리 울프가 심박수·수면시간을 그래프로 그려 자신을 이해하려 했다면, 만세력은 60갑자라는 시간축 위에 운기를 그린다. 도구는 다르지만 본질은 같다 — **자신의 패턴을 읽고 싶다는 인간의 원초적 욕망**.

### 2. "4행일기는 서울에서 쓴 명상록이다"
로마 황제가 전장에서 쓴 일기와, 2026년 한국의 직장인이 쓰는 4행일기. 시대도 지위도 다르지만 "오늘을 돌아보고 내일을 다짐한다"는 구조는 2000년 동안 변하지 않았다.

### 3. "탈레브의 경고를 아는 예측"
탈레브는 "과거 패턴으로 미래를 예측하는 건 위험하다"고 경고했다. 작가는 이 경고를 수용하면서도 한 발 나아간다 — 과거의 정합성을 먼저 검증하고, 신뢰도(%)를 붙여서 예측을 제시한다. 예측을 믿음이 아닌 참고선으로 쓴다.

### 핵심 해석

- **목화통명 (木火通明)**: 갑목이 병화를 생조, 인월에 태어난 병화 — 학식·표현력이 뛰어난 격국
- **일간 강왕**: 목생화로 일간이 매우 강함 → 토(土)가 용신으로 화기를 설기
- **금수 부재**: 재성(금)과 관성(수)이 없어 물질적 안정·조직적 권위보다 자기 표현·독립적 경로에 강점
- 태양(병화)의 성정 — 밝고 따뜻하지만, 혼자서 타오르는 존재

## 수비학 분석

### 생명수 계산

1979년 2월 18일:
- 1+9+7+9 = 26, 0+2 = 2, 1+8 = 9
- 26+2+9 = 37 → 3+7 = 10 → 1+0 = **1**

### 생명수 1 — 개척자 (The Pioneer)

- 리더십, 독립심, 창조력, 자기 주도
- 태양의 에너지 — 사주 일간 병화(丙火=태양)와 **정확히 일치**
- 스스로 길을 만드는 성향 — 조직보다 독립적 경로에서 빛남

### 글을 위한 핵심 인사이트

**사주와 수비학이 동일한 메시지를 전한다:**
- 사주: 병화(태양) + 목화통명 + 금수부재 = 독립적 창조자
- 수비학: 생명수 1 = 개척자
- 두 통계적 체계가 서로 다른 문명권에서 같은 결론 → "패턴이 보인다"는 주제의 강력한 근거

## 인생 그래프 시각화 개념 — "세계지도 같은 인생 지도"

### 비주얼 레퍼런스

실제 인생 그래프 예시 이미지 참조 (3개 레이어 선그래프):
- X축: 연도/나이
- Y축: 0점 기준 ± 점수
- 3개 색상 선이 독립적으로 파도침
- 고점 = 최고의 기회, 저점 = 최대 위기
- 4행일기 핀이 누적되며 실제 경험을 지도 위에 표시

### 비주얼 메타포: 세계지도

인생 그래프는 세계지도처럼 펼쳐진다:
- **가로축**: 나이/연도 (1979~2079)
- **세로축**: 0점 기준 ± (기회↑ / 위기↓)
- **3개 레이어**가 겹쳐 그려짐 (각각 다른 색상)
- 마치 **지형도의 등고선** — 취직은 산맥, 귀인은 강줄기, 건강은 기후선

### 3개 축 정의

| 축 | 사주 점수 | 의미 | 마이너스 예시 | 플러스 예시 |
|---|---|---|---|---|
| **취직(Job)** | 직업운·재성 | 경력, 이직, 창업, 실직 | 실직, 구조조정, 부도 | 취업, 승진, 직종 전환 성공 |
| **귀인(Gui)** | 귀인운·인연 | 사람, 기회, 만남 | 이별, 배신, 부친별세 | 귀인 조우, 해외취업, 협력 |
| **건강(Hlt)** | 건강운·질병 | 신체, 질병, 수술 | 심장수술, 입원 | 회복, 정화(사찰), 체력 회복 |

### 꼭지점 인터랙션 — 클릭 시 강도(%) + Top5 표시

그래프의 꼭지점(peak/trough)을 선택하면 해당 시점의 세부 강도와 Top5가 풍선/팝업으로 나타남:

```
         ★ 꼭지점 클릭
         │
    ┌────┴─────────────────────────┐
    │  2012년 7월 · 해외중국취업      │
    │                               │
    │  [ 강도 ]                      │
    │  취직(Job)    ████████░░ 82%   │
    │  귀인(Gui)    ██████░░░░ 63%   │
    │  애정(Love)   ███░░░░░░░ 28%   │
    │  건강(Hlt)    █████░░░░░ 51%   │
    │  재물(Wealth) ██████░░░░ 61%   │
    │                               │
    │  [ 과거 Top5 — 이 시점 기준 ]    │
    │  기회 1. 2000 첫 사회생활 (87%) │
    │  기회 2. 1998 대학입학   (82%) │
    │  위기 1. 1996 부도       (91%) │
    │  위기 2. 1990 정신충격   (78%) │
    │                               │
    │  [ 미래 Top5 — 이 시점 기준 ]    │
    │  기회 1. 2034 임신대운     (89%)│
    │  기회 2. 2044 계유대운     (85%)│
    │  위기 1. 2054 갑술대운     (52%)│
    │                               │
    │  📎 4행일기 2건 있음            │
    │  신뢰도: 87%                   │
    └────────────────────────────────┘
```

**팝업 3단 구성**:
1. **강도(%)**: 5개 지표(취직/귀인/애정/건강/재물)의 세부 강도 바
2. **과거 Top5**: 해당 꼭지점 시점을 기준으로 과거 기회·위기 Top5 (신뢰도% 표시)
3. **미래 Top5**: 해당 꼭지점 시점을 기준으로 미래 예측 기회·위기 Top5 (신뢰도% 표시)

꼭지점을 바꿀 때마다 Top5 기준점이 갱신됨 — 과거는 확정, 미래는 예측+신뢰도.

### 2단 구조: 기본 레이어 + 선택 레이어

**기본 레이어 (사주만으로 동작)**:
- 특정일만 입력(조회)해도 그래프 + 강도(%) + Top5 즉시 표시
- 사주 점수는 고정값이므로 4행일기 없이도 완전한 그래프가 그려짐
- 과거/미래 어떤 날짜든 조회 가능

**선택 레이어 (4행일기 누적)**:
- 4행일기는 선택적 추가 — 매일 쓸수록 세계지도 위에 **핀**이 꽂힘
- 핀 = 그 날의 실제 경험 (사실/느낌/교훈/선언)
- 핀이 선과 **정합**하면 → 신뢰도 상승, Top5 근거 강화
- 핀이 선과 **불일치**하면 → 예외 기록, 예측 보정에 활용
- 일기가 없는 꼭지점에는 "4행일기 없음" 표시, 신뢰도는 사주 점수만으로 산정

### 과거 Top5 / 미래 Top5 (각 축별)

각 축(취직/귀인/건강)마다 독립적으로:
- **과거 기회 Top5**: 실제 일기+사주 점수로 산출
- **과거 위기 Top5**: 실제 일기+사주 점수로 산출
- **미래 기회 Top5**: 사주 점수 + 과거 패턴 보정
- **미래 위기 Top5**: 사주 점수 + 과거 패턴 보정

예시 (작가의 과거 기준):
```
[취직 축 — 과거 위기 Top5]
1. 2025-09 구조조정 (경오운, 금화충돌) — 신뢰도 92%
2. 2005-09 두번째 퇴사 (기사운, 화토조열) — 신뢰도 85%
3. 2003-11 첫 퇴사 (무진운→기사운 전환기) — 신뢰도 78%
...

[건강 축 — 과거 위기 Top5]
1. 1983-08 심장판막증 수술 (정묘운, 목화극왕) — 신뢰도 95%
...
```
